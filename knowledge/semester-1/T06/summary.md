# T06 Summary: Portfolio Exam and Oral Presentation

## Main Focus

T06 is about the 1st semester portfolio exam.

The material focuses on:

- Portfolio website.
- Individual oral presentation.
- Exam requirements.
- Reflection on learning goals.
- Explaining process and decisions.
- Connecting projects to 1st semester learning.

## Portfolio Exam

The exam consists of:

- A portfolio website.
- An individual oral presentation.
- Follow-up examination based on the portfolio and presentation.

The purpose is to show the level of knowledge, skills, and competencies gained during 1st semester.

## What The Student Should Be Able To Explain

The student should be able to explain:

- What was built.
- Why it was built.
- Which design decisions were made.
- Which code decisions were made.
- Which methods were used.
- How user testing or feedback influenced the work.
- What worked well.
- What should be improved.
- How the work connects to learning goals.

## Useful Exam Explanation

I can explain my portfolio as evidence of what I learned in 1st semester. I should connect each project to methods, design choices, code choices, testing, and reflection instead of only showing the finished result.
