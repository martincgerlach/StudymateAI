# Semester 1 Knowledge Summary

## Purpose

This folder contains summarized knowledge from 1st semester Multimedia Design.

The raw source files are still kept in each theme folder. The Markdown summaries are written so StudyMate AI can use the knowledge without copying full teaching material into prompts.

## Theme Overview

| Theme | Focus | Useful for Assistants |
| --- | --- | --- |
| T01 | Intro project, title sequence, research, creativity, teamwork | Study Coach, UX Researcher, Project Manager, Exam Coach |
| T02 | Basic web, HTML, CSS, layout, visual hierarchy, design principles | Coding Mentor, Design Critic, Exam Coach |
| T03 | Research, user types, moodboards, lightning demos, testing, design brief | UX Researcher, Design Critic, Exam Coach |
| T04 | JavaScript basics, AI, information graphics, logo/vector work | Coding Mentor, Design Critic, Study Coach |
| T05 | Content production, visual branding, AI images, Photoshop, SCRUM, teamwork | Project Manager, UX Researcher, Design Critic, Exam Coach |
| T06 | Portfolio exam, portfolio website, oral presentation, reflection | Exam Coach, Study Coach, Project Manager |

## How StudyMate AI Should Use This

- Use T02 when the user asks about HTML, CSS, layout, hierarchy, images, classes, grid, fonts, or basic web structure.
- Use T03 when the user asks about research, interviews, user types, testing, surveys, moodboards, or design briefs.
- Use T04 when the user asks about JavaScript, DOM, events, AI in design, infographics, or logo/vector work.
- Use T05 when the user asks about content production, visual branding, Photoshop, AI image generation, SCRUM, or group planning.
- Use T06 when the user asks about exam preparation, portfolio explanation, oral presentation, or reflection.

## Current Limitation

These summaries are first-pass summaries based on the available semester material. They should be improved later with Martin's own project reflections, examples, and exam notes.
