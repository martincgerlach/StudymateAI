# T04 Summary: JavaScript, AI, Information Graphics, and Logo Work

## Main Focus

T04 introduced JavaScript basics and design work related to AI, information graphics, and vector/logo production.

The material focuses on:

- JavaScript in the browser.
- Connecting JavaScript files to HTML.
- `querySelector`.
- Events and event listeners.
- Changing content in the DOM.
- Console debugging.
- Information graphics and data visualization.
- AI-supported content/design work.
- Logo vector production.

## JavaScript Knowledge

JavaScript runs in the browser together with HTML and CSS.

Important beginner concepts:

- Add a script file with `<script src="js/main.js"></script>`.
- Use `document.querySelector()` to select an element.
- Use `.textContent` to change text.
- Use `addEventListener()` to react to user actions.
- Use `console.log()` to debug in the browser console.

Example concept:

```js
const headline = document.querySelector("h1");
headline.addEventListener("click", changeColor);
```

## Information Graphics

Information graphics and data visualization help communicate information faster and more clearly than text alone.

Important ideas:

- Visual communication can make complex information easier to understand.
- Good infographics need clear hierarchy and simple visual structure.
- The design should support the data, not decorate it randomly.

## Logo and Vector Work

The logo material focuses on creating clean vector logos.

Important ideas:

- Use proper vector shapes.
- Avoid messy auto-traced artwork.
- Keep anchor points controlled.
- Convert text to outlines when needed.
- Aim for a clean scalable logo file.
