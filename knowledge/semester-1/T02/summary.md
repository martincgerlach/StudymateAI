# T02 Summary: Basic Web, HTML, CSS, Layout, and Visual Hierarchy

## Main Focus

T02 introduced basic web development and web design.

The material focuses on:

- HTML structure.
- Semantic elements.
- Images and alt text.
- CSS basics.
- Classes.
- Simple layout.
- Responsive thinking.
- Visual hierarchy.
- Gestalt principles.
- Call to action design.
- Fonts and fallback fonts.
- File and folder structure.
- GitHub and basic web workflow.

## HTML Knowledge

HTML is used for structure, not visual styling.

Important ideas:

- Use heading levels to create hierarchy.
- Use paragraphs for text sections.
- Use semantic elements when they describe the content.
- Use `div` only when no semantic element fits.
- Use `img` with `src` and useful `alt` text.
- Use correct file paths so the browser can find assets.

## CSS Knowledge

CSS controls the visual presentation.

Important ideas:

- Use classes to reuse styling.
- Make images responsive with `display: block` and `max-width: 100%`.
- Use hover and focus states for interactive elements.
- Use clear spacing, color, typography, and hierarchy.
- Validate CSS when debugging.

## Design Knowledge

The design material connects web layout to visual communication.

Important ideas:

- Visual hierarchy helps users understand what is most important.
- Contrast, size, position, and color can guide attention.
- Gestalt principles help group related information.
- CTA buttons should stand out clearly from normal content.
- Typography should use readable fonts and fallback font stacks.

## Useful Exam Explanation

I can explain that HTML creates the structure, CSS creates the visual design, and responsive layout helps the website work on different screens. I can also explain design choices using hierarchy, contrast, spacing, Gestalt principles, and CTA clarity.
