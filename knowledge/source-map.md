# StudyMate AI Source Map

## Purpose

This document tracks where the knowledge inside the StudyMate AI Knowledge Base originates from.

Its purpose is to:

- Prevent duplicate knowledge.
- Keep references to original projects.
- Track imported learning material.
- Help maintain the Knowledge Base over time.
- Make it easy to update knowledge when projects change.

The source projects are never modified. StudyMate AI only reads them and creates summarized knowledge files.

## Import Workflow

Every new project should follow this process:

- Project is completed.
- Original project remains untouched.
- StudyMate AI scans the project.
- Knowledge is summarized.
- A markdown knowledge file is created.
- Source Map is updated.
- The Knowledge Loader can use the new knowledge.

## Rules

- Never copy entire projects into the Knowledge Base.
- Never duplicate source code.
- Never import secrets, passwords, API keys, or private files.
- Always summarize knowledge.
- Always keep the original project as the source of truth.
- Update this document whenever a new project is imported.
